Это проект для распознавания парковочных мест
Нажав на кнопку print можно распечатать коды для парковочных мест
Можно использовать сетевую камеру (проверял на камере cisco) или обычную вебку
Принцип такой: за каждым кодом закреплено определенное место (0-9) если камера видит определенный код значит это место свободно, иначе занято.
В настройках программы можно указать какие места в поле зрения камеры.
Недостаток тут такой что если код перекрыт чем либо (грязью например) то место будет считаться занятым
Программа с определенной периодичностью обновляет состояние мест в базе mongo sp_db прописано хардкодно
