Проект может работать с сетевой камерой, проверялось на какой то cisco, или с обычной веб камерой в настройках (кнопка настройки) можно выбирать камеру.
Программа посылает номер не сразу, а когда получит несколько результатов распознавания (10 хардкод).
Отправление номера происходит в методе bool sendNumber(string number), результат распознавания отправляется по адресу http://localhost:8080/api/parking/auto (тоже хардкод, при необходимости поменять)
Для тестирования распознавания с камеры можно напечаать автомобильные номера, ну или просто на смартфоне подносить к камере.
Есть еще кнопка "..." для выбора изображения с автомобильным номером, нужно скорее для теста.
